# PDF Edit

一个基于 Web 的 PDF 编辑器，支持批注、文本编辑、表单填写、批改模式和 AI 智能批改。

## 功能特性

- **PDF 查看**：基于 pdf.js 渲染，支持缩放、翻页
- **批注工具**：矩形高亮、箭头、文字批注、画笔（原生 Canvas 2D）
- **文本编辑**：直接点击 PDF 文字进行在线修改
- **表单填写**：自动识别 PDF 表单字段并填写
- **页面操作**：旋转、删除页面
- **批改模式**：左右分栏对比答案和作业，独立翻页
- **AI 智能批改**：接入 OpenAI / Claude，自动评分并生成评语
- **AI 自动补全**：文字批注时 AI 续写辅助
- **搜索高亮**：跨页面搜索文本并高亮定位
- **多格式支持**：PDF、Word (.docx) 文件
- **临时保存**：刷新后自动恢复工作状态

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS
- pdf.js（PDF 渲染）
- pdf-lib（PDF 操作）
- mammoth.js（DOCX 转换）

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 部署

### 1Panel 部署

1. 创建网站，反向代理到 `http://127.0.0.1:5173`
2. 在项目目录运行 `npm run build`
3. 使用 `npm run preview` 或 `npx serve dist` 启动服务

### 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `PORT` | 服务端口 | 5173 |

## AI 功能配置

1. 点击 Toolbar 上的「AI 设置」⚙️
2. 选择服务商（OpenAI / Claude）
3. 选择或输入模型名称（支持自定义模型）
4. 填入 API Key
5. 保存后即可使用 AI 批改和 AI 续写

> API Key 仅保存在浏览器本地，不会上传到服务器。

## 使用说明

### 基本操作

| 功能 | 操作 |
|------|------|
| 上传文件 | 点击「打开 PDF」或拖拽文件 |
| 翻页 | 点击 ← → 按钮或输入页码 |
| 缩放 | 点击 − / + 按钮 |
| 批注 | 切换到「批注」模式，选择工具后拖拽绘制 |
| 文字批注 | 选择「文字」工具，点击位置输入内容 |
| AI 续写 | 文字批注输入框中点击「AI」按钮 |
| 删除批注 | 选择工具点击批注，按 Delete 键 |
| 文本编辑 | 切换到「文本」模式，直接点击文字修改 |
| 表单填写 | 切换到「表单」模式，在右侧面板填写 |

### 批改模式

1. 点击「批改」按钮进入分栏模式
2. 左侧加载答案 PDF，右侧为作业 PDF
3. 可使用「自动比对」进行简单文本差异标记
4. 或使用「AI 批改」进行智能评分

### 搜索

- 在 Toolbar 搜索框输入关键词
- 点击 🔍 搜索所有页面
- 使用 ↑↓ 在结果间跳转
